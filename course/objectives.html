    Section Specific Learning Objectives for Each Topic

    The objectives below are organized according to the sections of Active Prelude to Calculus, by Matthew Boelkins. Note that italicized words and phrases are terminology that you are expected to know, understand, and be able to work with.
    🔗

    Subsection Relating Changing Quantities (Chapter 1)

    Subsubsection Changing in Tandem (Section 1.1)

    You should be able to do the following:
    • Given two related quantities and appropriate information about the relationship between the quantities, describe how one quantity changes when the other changes.
      🔗
      🔗
    • Draw a rough graph of the relationship between two given variables.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Functions: Modeling Relationships (Sections 1.2)

    You should be able to do the following:
    • Explain how a function is a process that converts inputs into outputs.
      🔗
      🔗
    • Explain the definition of a function.
      🔗
      🔗
    • Determine whether or not a given relation is a function.
      🔗
      🔗
    • Determine the domain, codomain, and range of a given function.
      🔗
      🔗
    • For an applied problem, determine the domain and range of a function that makes sense for the problem (e.g. if the input is distance, it cannot be a negative number).
      🔗
      🔗
    • For a function given as a relation, find the formula for the function.
      🔗
      🔗
    🔗
    🔗

    Subsubsection The Average Rate of Change of a Function (Section 1.3)

    You should be able to do the following:
    • Given a function \(f\) and an interval \([a, b]\) in the domain of \(f\text{,}\) compute the average rate of change over the interval (i.e. compute \(\frac{f(b)-f(a)}{b-a}\)).
      🔗
      🔗
    • In an applied problem, interpret what the average rate of change of the function means (e.g. if the input \(x\) is time and \(f(x)\) is the distance traveled at time \(x\text{,}\) then the average rate of change is velocity).
      🔗
      🔗
    • In an applied problem, identify the units of the average rate of change (e.g. if the input \(x\) is time given in minutes and the function \(f(x)\) is the temperature given in \(^\circ\)F of an object at time \(x\) minutes, then the units of the average rate of change is \(^\circ\)F per minute).
      🔗
      🔗
    • Use the average rate of change to make predictions about the function.
      🔗
      🔗
    • Identify when a function is increasing and when it is decreasing.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Linear Functions (Section 1.4)

    You should be able to do the following:
    • Recognize the formula of a linear function (i.e. \(f(x) = ax + b\text{,}\) where \(a\) and \(b\) are constants).
      🔗
      🔗
    • Interpret a linear function as a function where the average rate of change between every pair of points is the same.
      🔗
      🔗
    • Given a point on a line and the slope of the line, determine a formula for the linear function.
      🔗
      🔗
    • Given two points on a line, determine a formula for the linear function.
      🔗
      🔗
    • For an applied problem involving a linear function, use your understanding of linear functions to interpret the function and make predictions.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Quadratic Functions (Section 1.5)

    You should be able to do the following:
    • Recognize a quadratic function in both standard form and vertex form.
      🔗
      🔗
    • Given a quadratic function in any form, determine the vertex.
      🔗
      🔗
    • Given three points on the graph of a quadratic function, determine the system of equations necessary to find the formula for the quadratic function.
      🔗
      🔗
    • Given a quadratic function in any form, determine at what values of \(x\) the function is zero. This includes factoring methods and the quadratic formula.
      🔗
      🔗
    • Given a quadratic function, determine whether it opens up or opens down.
      🔗
      🔗
    • Given an applied problem involving a quadratic function, use your understanding of quadratic functions to interpret the function and make predictions.
      🔗
      🔗
    • Use solution methods of quadratic equations to solve applied problems involving quadratic functions.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Composite Functions (Section 1.6)

    You should be able to do the following:
    • Given formulas for two functions \(f\) and \(g\text{,}\) determine a formula for the function \(f \circ g\text{.}\)
      🔗
      🔗
    • Given the graph of two functions \(f\) and \(g\text{,}\) compute \((f \circ g)(x)\) for given values of \(x\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Subsubsection Inverse Functions (Section 1.7)

    You should be able to do the following:
    • Explain and use the definition of inverse functions.
      🔗
      🔗
    • Given the formula for a function \(f\text{,}\) determine the formula for the inverse function \(f^{-1}\text{.}\)
      🔗
      🔗
    • Given the graph of a function \(f\text{,}\) determine whether or not the inverse function \(f^{-1}\) exists.
      🔗
      🔗
    • Given the graph of a function \(f\text{,}\) sketch the graph of \(f^{-1}\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Subsubsection Combining Functions (Section 1.9)

    You should be able to do the following:
    • Solve application problems where functions are created by adding, subtracting, multiplying, and dividing other functions.
      🔗
      🔗
    • Determine the formulas and input values for a piecewise defined function, given appropriate information about the function.
      🔗
      🔗
    • Sketch the graph of a piecewise defined function.
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection Exponential and Logarithmic Functions (Chapter 3)

    Subsubsection Exponential Growth and Decay (Section 3.1)

    You should be able to do the following:
    • Explain how an exponential function describes a quantity that either increases or decreases by a fixed percentage over some constant change in the input values (e.g. a quantity that increases by 5% every 2 years).
      🔗
      🔗
    • Determine the formula for an exponential function given information about the percent increase/decrease of the function and the value of the function at a given input.
      🔗
      🔗
    • Determine the formula for an exponential function given the values of the function at two given inputs.
      🔗
      🔗
    • Use your understanding of exponential functions to solve application problems (e.g. interest rate [including compound interest] & radioactive decay).
      🔗
      🔗
    🔗
    🔗

    Subsubsection Modeling with Exponential Functions (Section 3.2)

    You should be able to do the following:
    • Describe the end behavior of the function \(f(x) = ab^x\) for different values of \(b\) (i.e. what happens to the function as \(x \rightarrow \infty\) or \(x \rightarrow \infty\)).
      🔗
      🔗
    • Explain why a vertical translation (and perhaps reflection) of an exponential function is appropriate for the temperature of an object over time.
      🔗
      🔗
    • Given appropriate information about the temperature of an object and its environment, construct the formula for a function that models the temperature of the object.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Compound Interest & the Natural Exponential Function (Section 3.3)

    You should be able to do the following:
    • Solve problems involving compound interest.
      🔗
      🔗
    • Know that the function \(f(x) = e^x\) is the natural exponential function.
      🔗
      🔗
    🔗
    🔗

    Subsubsection What a Logarithm Is (Section 3.4)

    You should be able to do the following:
    • Explain why the function \(f(x) = b^x\) has an inverse when \(b > 0\) and \(b \ne 1\text{.}\)
      🔗
      🔗
    • Know that the inverse of \(f(x)\) in this case is \(f^{-1}(x) = \log_b(x)\text{.}\)
      🔗
      🔗
    • Use the definition of the logarithm functions to compute \(\log_b(x)\) for certain values of \(x\) and \(b\text{.}\)
      🔗
      🔗
    • Use your understanding of logarithms to solve equations involving exponential and logarithm functions.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Properties and Applications of Logarithmic Functions (Section 3.5)

    You should be able to do the following:
    • Explain the properties \(\log_b(xy) = \log_b(x) + \log_b(y)\text{,}\) \(\log_b \left( \frac{x}{y} \right)= \log_b(x) - \log_b(y)\text{,}\) and \(\log_b(x^r) = r \log_b(x)\text{.}\)
      🔗
      🔗
    • Use the above properties to solve equations involving variables in exponents.
      🔗
      🔗
    • Use the above properties to solve applied problems involving exponential functions.
      🔗
      🔗
    • Apply your understanding of exponential and logarithm functions to solve problems similar to half-life problems for exponential decay.
      🔗
      🔗
    • Apply your understanding of exponential and logarithm functions to solve problems similar to doubling time problems for exponential growth.
      🔗
      🔗
    • Use your understanding of exponential functions and logarithm functions to solve applied problems involving temperature of objects in an environment of constant temperature.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Modeling Temperature and Population (Section 3.6)

    You should be able to do the following:
    • Use your understanding of exponential functions and logarithm functions to solve applied problems involving exponential and logistic population models.
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection Data Analysis

    Subsubsection Regression Models

    You should be able to do the following:
    • Use Desmos or other technology to represent a given data set as points on a graph.
      🔗
      🔗
    • Determine from the graph of the data whether the relationship between the quantities given by your data set is linear, quadratic, exponential, logarithmic, vertically translated exponential (i.e. temperature model), or logistic.
      🔗
      🔗
    • Use Desmos or other technology to find an appropriate regression model (i.e. function) on your data.
      🔗
      🔗
    • Analyze the value of \(R^2\) and the appearance of the graph of the regression model to determine how well your model fits the data.
      🔗
      🔗
    • Use your regression model to make predictions about the quantities measured by your data set.
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection Polynomial and Rational Functions (Chapter 5)

    Subsubsection Power Functions & Polynomials (Sections 5.1 & 5.2)

    You should be able to do the following:
    • Recognize what a polynomial function \(f(x)\) approaches as \(x \rightarrow \infty\text{.}\)
      🔗
      🔗
    • Recognize what a polynomial function \(f(x)\) approaches as \(x \rightarrow -\infty\text{.}\)
      🔗
      🔗
    • Identify all \(x\)-intercepts of the graph of a polynomial expressed in factored form.
      🔗
      🔗
    • Use the factored form of a polynomial to determine how the graph of the function behaves around the \(x\)-intercepts of the graph of the polynomial (i.e. does the graph “touch” the \(x\)-axis or “cross” the \(x\)-axis).
      🔗
      🔗
    • Given the graph of a polynomial function, derive a plausible formula for the function.
      🔗
      🔗
    🔗
    🔗

    Subsubsection Modeling with Polynomial Functions (Section 5.3)

    You should be able to do the following:
    • For a quantity that depends on two variables and also has a constraint, express the quantity as a polynomial or rational function of one variable.
      🔗
      🔗
